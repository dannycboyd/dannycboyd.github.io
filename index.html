Internal Displacement in Post-2015 Burundi

CMPS 165 final project proposal

Danny Boyd

This project will aim to provide a clear and succinct look at numbers of internally displaced people in Burundi after the 2015 coup and following political unrest. This project will extend my Program 6, which did this for one snapshot in jan 2018. I'm curious in particular to see how the data changes as time progresses. My proposal then is a map, similar to my Program 6, but extended with a time slider so that the other reports will be visualized.

The displacement reports, which are the bulk of the data for this project, will come from the Displacement Tracking Matrix, a project started by the International Organization for Migration (IOM). They have compiled comprehensive statistics on many aspects of the displacement. For example, here are some of the visualisations which the DTM has put together for 2017

  • Burundi 2017 Dashboards
  • Burundi 2018 Dashboards

Another visualization, not of displacement but of violent events is this one, assembled by the Armed Conflict Location & Event Data Project (ACLED). The data tracks the number of individual conflict events over time, the number and location of violent events as well as what type of violence it was, and the number of conflicts commited by various armed groups

The United Nations Refugee Agency also has assembled a report on Burundi, which can be found here. Details include externally displaced people, as well as plans for strategy and coordination
